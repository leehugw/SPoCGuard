# SPoCGuard

SPoCGuard is a hybrid Neuro-Symbolic smart contract security analysis framework designed to detect vulnerabilities in Solidity source code. The framework combines static analysis, symbolic reasoning, and large language model capabilities to improve vulnerability detection accuracy while autonomously generating compilation-ready Proof-of-Concept scaffolds.

## Prerequisites

Python **3.10 or later** is required.

## Environment Setup

Create a virtual environment:

```bash
python -m venv env
```

Activate the virtual environment.

### Windows

```bash
env\Scripts\activate
```

### Linux / macOS

```bash
source env/bin/activate
```

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Install and configure Solidity compiler version 0.8.20:

```bash
solc-select install 0.8.20
solc-select use 0.8.20
```

## Project Structure

Place the Solidity source code to be analyzed inside:

```text
src/sourcecode/
```

Example:

```text
src/
├── sourcecode/
│   └── MyContract.sol
├── main.py
```

## Usage

Run the framework:

```bash
python src/main.py
```

## Output

After running SPoCGuard, the generated results are stored in the `res/` directory:

```text
SPOCGUARD/
├── src/
├── res/
│   ├── detection_report.json
│   ├── PoC/
│   └── halmos_test/
```

- `detection_report.json`: Final vulnerability detection report in JSON format.
- `PoC/`: Generated Proof-of-Concept files for detected vulnerabilities.
- `halmos_test/`: Test cases automatically generated by Halmos during symbolic analysis.
